For loops have three parts: the initalization, test, and increment.

for (int a = 1; a < 10; a++){
    System.out.println(a);
}

While loops are pretty similar to while loops in Python.
int a = 1;
while (a < 10) {
    System.out.println(a);
    a++;
}
